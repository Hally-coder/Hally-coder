# Clario — AI Personal Finance Coach

Clario is an AI-powered personal finance application designed for students, young earners, gig workers, and busy professionals. It turns financial activity into simple, practical guidance—helping users understand spending, control budgets, plan savings, and take realistic financial actions.

> The production source code is private. This page is a code-free product showcase.

## Key features

- Income, expense, balance, and monthly budget tracking
- Manual transaction entry and CSV import
- Optional bank-account synchronization
- Automatic expense categorization
- Recurring subscription detection
- Cash-flow forecasting
- Weekly AI-generated financial coaching
- Savings goals and contribution tracking
- Spending-reduction scenario planning
- Account data export and deletion controls
- Installable Progressive Web App experience

## Product approach

Clario is designed as a supportive money coach rather than a traditional banking dashboard. It focuses on clear, specific, non-judgmental recommendations that users can realistically follow.

## Technology overview

- Next.js, React, and TypeScript
- Prisma and relational data storage
- Secure authentication
- AI-assisted coaching with a rules-based fallback
- Bank-linking integration
- Responsive financial visualizations
- Vercel-ready deployment

## Engineering highlights

- Accurate integer-based money calculations
- User-scoped data access
- Encrypted handling of sensitive connection data
- Anonymized financial context for AI coaching
- Graceful fallback when an AI provider is unavailable
- Production-ready database migration path

## Status

Clario is under active development. Its implementation is maintained in a private repository to protect the source code and security-sensitive configuration.
